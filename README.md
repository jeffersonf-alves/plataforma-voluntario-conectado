
## Voluntário Conectado

Uma plataforma de contribuição sem fins lucrativos, o objetivo é que instituições do terceiro setor possam cadastrar suas vagas temporárias para receber candidatos dispostos a oferecer seus serviços de forma gratuita.

## Tecnologias Utilizadas
- Java 
- Spring boot, JPA e Security
- Postgres SQL
- Vue.js
